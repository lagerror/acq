# 关于传统图书馆一些胡话
## 编目 像京东图书那样尽可能的多展示一些有用内容，如编辑推荐 前言 目录  试读 点评 相关资源等，建立一个阅读社群不断进化；
## 流通 有些公共图书馆已经在外包了，而这个趋势正在扩大；
## 典藏 更多的是共建共享，一般学校的以一馆之力做不到馆藏体系建设和学科资源保障，把纸质图书的典藏交给大馆和国家版本图书馆；
## 图书共享 等待基于读者自购的图书借阅量大于了图书馆馆藏的借阅量；
## 阅读推广 以一类书 一类人建立学习社群，提供名师，资源和学习评价等功能；
